# Mélo Beauty Lounge — Website

Public marketing website for Mélo Beauty Lounge, a ladies-only beauty salon in Barbar, Bahrain.

## Dev

```bash
npm install
npm run dev     # http://localhost:3000
npm run build
npm run lint
```

## Stack

- Next.js 14 (App Router, static export)
- TypeScript
- Tailwind CSS
- Framer Motion
